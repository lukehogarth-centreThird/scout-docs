Centre Third Scout

1 Installation

  1. Download the .dmg file from the web Dropbox folder that has been shared with you.
  2. Double-click on the .dmg file and drag and drop the 'centrethird-scout' icon into you Applications folder.
  3. After installation, the centrethird-scout app will be available in your Applications.

2 Import session data

  1. Download the session folder from web Dropbox folder that has been shared with you.
  2. Double-click the downloaded .zip file to expand the folder.
  3. Go into the expanded folder where you will see a sessionId folder. Don't change the name of this folder.
  4. Move or copy the sessionId folder into 'Documents -> netballR -> data -> Local -> sessionData'
  5. Close and reopen the centrethird-scout app, or in the taskbar go to 'View' -> 'Reload'. The imported session data will now be available in the 'Database' tab.

3 How to use the app

3.1 How to load session video and data

  1. Open Centre Third Scout and you will be on the 'Database' tab.
  2. Select a session using the check boxes on the left hand side of the table.
  3. You can use the 'Delete' or 'Load' buttons when one or more sessions are selected using the check boxes.
  4. Use the table header to search and filter for session by competition, round or team name.
  5. Select a session and click 'Load' to import that sessions video and data into the 'Scout' tab.
  6. There are two templates that you can choose from the dropdown select to load data - 'live' or 'video'.
  7. The 'video' template loads data that has been collected post-match and will usually be the only data availalbe for opposition matches.

3.2 How to use the scout tab

  1. Navigate to the 'Scout' tab using the top navbar.
  2. There are three main panels. The top right container is the playlist data. The top left container is the video player and match reports.
  3. Use the buttons in container headers to expand or minimise their size. The tab panels on the left hand side of container headers can be used to move between the nested tab panels within containers.
  4. The video player control tools at the bottom of the tab can be used to:
    • view multiple camera angles,
    • jump backward or forward, play, pause or seek on the timeline,
    • play the camera angle in picture-in-picture mode watch footage in an extended display,
    • control the playback speed of the video,
    • and create clips from the current video time that can be added to the draft playlist.
  5. The bottom container includes the timeline and rowwise event data in seperate tabs. Double-click on the buttons in the timeline to search on the video. Pause the video and click on a timeline button while holding 'command' to select and add to a draft playlist.
  6. You can also create playlists using the rowwise data in the other tab panels. Use the table headers to filter tables and select data using the check boxes to create a new draft playlist or add selected rows to an existing draft playlist.
  7. The draft playlist can then be saved, used to search events on the video, or to export clips as an .mp4 file

3.3 How to create a playlist and export clips

  1. You can create a draft playlist by using the 'Clip' button in the video player or by using the timeline and rowwise event data in the bottom container.
  2. Add any notes or tags to the clip, and change the 'lead time' to search the start the clip. Hit the 'add to draft playlist' button to move the clip into the playlist container. You can then export the clip as an mp4 or continue adding clips to the draft playlist.
  3. You can also use the rowwise table data to search and filter events using the table headers.
  4. The draft playlist can then be used to search each event in the video by selecting the table row.
  5. You can save the draft playlist to view later or to export individual clips in the playlist.
  6. Select the saved playlist and hit 'Move to draft' to view or export a single clip in the playlist.
  7. Select the saved playlist and hit 'Export clip' to export a zip folder containing all clips in the playlist.
